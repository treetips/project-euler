# Project Euler for TypeScript

## Usage

```sh
# Git clone
git clone git@github.com:treetips/project-euler.git
cd project-euler/typescript
npm i
# Run all unit tests.
./unit-test.sh
# Run one unit test.
./unit-test.sh p001
```
